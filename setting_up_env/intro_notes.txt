FastAPI is a modern Python framework for building APIs.
It is designed for speed, validation, and developer productivity.
It uses Starlette for web functionality.
It uses Pydantic for data validation.
It's an excellent choice for AI/ML backends and microservices.
______________________________________________________________
When to Use Each?

--> Choose FastAPI when:

- Building REST APIs
- AI/ML services
- LLM applications
- Microservices
- High-performance backends

--> Choose Flask when:

- Learning web basics
- Very small projects
- Simple prototypes

--> Choose Django when:

- Large web applications
- Admin dashboards
- CMS-like systems
- Projects needing batteries-included features
